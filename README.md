# Amazon Homepage Clone

A frontend clone of Amazon's homepage built using pure HTML and CSS.

## 🖥️ Live Demo
[Click here to view live](https://vidhya-majee.github.io/amazon-clone)

## 📸 Features
- Responsive Navbar with search bar and icons
- Hero/Banner section
- Product showcase sections
- Hover effects on products and buttons
- Footer with multiple columns

## 🛠️ Tech Stack
- HTML5
- CSS3

## 📂 How to Run Locally
1. Clone the repo:
   git clone https://github.com/Vidhya-Majee/amazon-clone.git
2. Open index.html in your browser
